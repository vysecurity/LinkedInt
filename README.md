# Usage

1. Put in LinkedIn credentials in LinkedInt.cfg
2. Put Hunter.io API key in LinkedInt.cfg
3. Run `pipenv shell` to activate virtual environment
4. Run `python LinkedInt.py` and follow instructions (example below).
